# Blockchain-IoT

A majority of the web relies on centralized certificate authorities to authenticate new and existing endpoints. This works in practice, but is subject to breaches and compromises that affect thousands of clients. 

We propose an isolated, self-authenticating network that avoids relying on a centralized external authority for authenticating previously identified devices. By using a distributed, locally stored blockchain, we establish a decentralized, immutable trust system among devices on the network.

While we recognize the limitation that new, unencountered devices must be authenticated using a centralized system, we verify that a decentralized system is both possible and practical to reduce dependence on a central authority for protecting existing devices.
